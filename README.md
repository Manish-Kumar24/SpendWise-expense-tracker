# SpendWise — Expense Tracker

A personal expense tracking app with dashboard, analytics & budgeting.

## 🚀 Features
- ➕ Add income/expense transactions
- 📊 Visual spending breakdown (charts)
- 💰 Set category budgets with progress tracking
- 🔍 Search & filter transactions
- 💾 Data saved locally in your browser

## 🛠️ Tech Stack
- HTML5, CSS3 (CSS Variables), Vanilla JavaScript
- localStorage for data persistence
- Deployed on Vercel
