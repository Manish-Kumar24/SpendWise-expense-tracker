# 💰 SpendWise — Expense Tracker

A clean, modern personal expense tracker with dashboard analytics, category budgets, and local data persistence.

![SpendWise Dashboard](https://via.placeholder.com/800x400/07111d/5baaee?text=SpendWise+Preview)

## 🚀 Live Demo

🔗 [https://spend-wise-expense-tracker-chi.vercel.app](https://spend-wise-expense-tracker-chi.vercel.app)

## ✨ Features

- ➕ **Add Transactions**: Record income & expenses with date, category & notes
- 📊 **Dashboard**: Visual summary of income, expenses & net balance
- 🥧 **Analytics**: Category breakdown with bar charts & donut visualization
- 📈 **Trends**: Monthly spending trend line graph
- 💰 **Budgets**: Set monthly limits per category with progress tracking
- 🔍 **Search & Filter**: Find transactions by description, category or type
- 💾 **Local Storage**: All data saved in your browser — no account needed
- 🌙 **Dark UI**: Modern, eye-friendly interface with smooth animations
